# Patient Registration System

This project is a simple web application developed in Java with a relational database.

## Features
- Patient registration
- Data storage in database
- Basic data management

## Technologies
- Java
- SQL
- Server Payara

## Description
The application allows users to register patients and manage their data using a relational database.

## How to run
1. Clone the repository
2. Open the project in your IDE (e.g., IntelliJ / NetBeans)
3. Configure database connection
4. Run the application
